== Description ==

O **WP Admin Style UI** foi desenvolvido para auxiliar os desenvolvedores iniciantes ou veteranos a criar tela de administração WordPress nativa, utilizando ferramentas e classes, sem a necessidade de importar ou adicionar arquivos externos.

== Installation ==

A instalação é fácil e rápida, tudo é realizado pelo painel do WordPress.


== Frequently Asked Questions ==

= O que é WP Admin Style UI? =

Um plugin que deverá auxiliar desenvolvedores a integrar com a interface de administração do WordPress.

= Como utilizar? =

Instale o plugin normalmente baixando ou utilizando a interface do WordPress, com o plugin ativado, acesso no menu do WordPress **Admin Style UI** e você terá uma boa interface para entender e estudar como funciona.
